# November 2023 release

#### ✅ Release Date: November 26, 2023

#### ✅ Version 1.0.0

`Ver 1.0.0` 첫 릴리즈 노트입니다. 다음과 같은 기능이 있습니다.

## 📝 설명

### ✅ 로그인

✓ `New` : 이름을 입력하면, localstorage 의 유저 정보와 비교하여 로그인하는 기능 구현

### ✅ 전체 기록

✓ `New` : 수업 시간에 따라 전체 기록 노출 기능 구현

### ✅ 개인 기록

✓ `New` : 개인 기록 노출 기능 구현 & 전체 기록에 자신의 기록 노출 통제 기능 구현


### ✅ 익명 게시판

✓ `New` : 익명으로 게시글을 남기는 기능 구현
